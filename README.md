## 使用方式

### 1. 下载配置文件

![下载](https://i.loli.net/2019/12/06/3VBawUXLrsMR8d9.jpg)

点击下载，将下边的链接拷贝到输入框。

https://raw.githubusercontent.com/alpha87/QuantumultX-Profiles/master/quantumultX_profile.conf

保存。

### 2. 导入节点

可以手动导入或者订阅链接导入。
